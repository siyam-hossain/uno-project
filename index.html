Overview

FireBot R1 is an autonomous ground vehicle that detects, approaches, and extinguishes small fires. It uses three IR flame sensors arranged left-center-right to triangulate fire direction, an HC-SR04 ultrasonic sensor (mounted on a pan servo) for distance measurement and obstacle avoidance, and a relay-controlled water pump with an aim servo to direct the nozzle.

How it works: The robot continuously polls its flame sensors. When fire is detected, it navigates toward the source while avoiding obstacles. Within 15 cm, it stops, aims the nozzle, and activates the pump. If no fire is present, all systems idle.

All pin assignments are defined at the top of the sketch as const int constants, making them easy to remap for different chassis configurations.

Setup Function

Runs once on power-on or reset. Initializes all peripherals, centers both servos, and ensures the pump is off before entering the main loop.

void setup() {
  Serial.begin(9600);

  // Motor driver pins
  pinMode(IN1, OUTPUT); pinMode(IN2, OUTPUT);
  pinMode(IN3, OUTPUT); pinMode(IN4, OUTPUT);
  pinMode(ENA, OUTPUT); pinMode(ENB, OUTPUT);

  // Ultrasonic + pump
  pinMode(TRIG, OUTPUT); pinMode(ECHO, INPUT);
  pinMode(PUMP, OUTPUT);

  pumpOff();               // Safe state first
  panServo.attach(SERVO_PAN);
  aimServo.attach(SERVO_AIM);
  panServo.write(90);      // Center pan servo
  aimServo.write(90);      // Center aim servo

  stopMotors();
  delay(1000);
  Serial.println("Fire Fighting Robot Ready");
}

Main Loop Logic

The loop runs continuously. Each iteration reads all sensors and makes a single navigation/action decision based on a priority chain:

1
No fire detected

All sensors above FLAME_THRESH → pump off, motors stop, return early.

2
Fire detected, within extinguish range

Distance ≤ EXTINGUISH_CM (15 cm) → stop, aim nozzle, pump on.

3
Obstacle detected

Distance < OBSTACLE_CM (20 cm) → pump off, run avoidance maneuver.

4
Navigate toward fire

Center sensor → forward. Left only → turn left. Right only → turn right. Both sides → forward.

void loop() {
  bool fL = (analogRead(FLAME_L) < FLAME_THRESH);
  bool fC = (analogRead(FLAME_C) < FLAME_THRESH);
  bool fR = (analogRead(FLAME_R) < FLAME_THRESH);
  long distance = getDistance();

  if (!fL && !fC && !fR) { pumpOff(); stopMotors(); return; }

  if (distance <= EXTINGUISH_CM) {
    stopMotors(); aimNozzle(fL, fC, fR); pumpOn(); return;
  }

  if (distance < OBSTACLE_CM) { pumpOff(); avoidObstacle(); return; }

  pumpOff();
  if (fC || (fL && fR))  moveForward();
  else if (fL)           turnLeft();
  else if (fR)           turnRight();
}

Ultrasonic Sensor

The getDistance() function sends a 10 µs pulse on TRIG and measures the echo duration. Distance is computed as duration / 58 (cm). A 25 ms timeout prevents blocking if no echo returns, in which case 999 cm is returned.

long getDistance() {
  digitalWrite(TRIG, LOW);  delayMicroseconds(2);
  digitalWrite(TRIG, HIGH); delayMicroseconds(10);
  digitalWrite(TRIG, LOW);

  long duration = pulseIn(ECHO, HIGH, 25000);
  return (duration == 0) ? 999 : duration / 58;
}
Formula: Sound travels ~343 m/s at room temperature. The round-trip distance in cm = (duration in µs) / 58.

Obstacle Avoidance

When an obstacle is closer than OBSTACLE_CM, the robot executes a 4-step avoidance maneuver using the pan servo to scan left and right before choosing a direction.

1
Stop — halt motors, 150 ms settle.
2
Scan left — pan servo to 160°, measure distance.
3
Scan right — pan servo to 20°, measure distance. Return to 90°.
4
Reverse — back up for BACK_MS ms, then turn toward the clearer side for TURN_MS ms.

Fire Detection

Three IR flame sensors return analog values (0–1023). A lower value indicates a stronger IR signal — i.e., more flame. The threshold FLAME_THRESH = 500 means any reading below 500 is treated as fire detected on that side.

Sensor Placement: Mount sensors angled slightly outward at roughly 45° increments for good lateral coverage. Keep them away from direct sunlight, incandescent bulbs, and hot surfaces to avoid false positives.

Nozzle Aiming

The aimNozzle() function positions the aim servo based on which flame sensors are active. This only runs when the robot is within extinguish range.

Pump Control

The pump is driven by a relay module. The RELAY_ACTIVE_LOW flag controls signal polarity — most common relay breakout boards are active-low (writing LOW activates the relay).

void pumpOn() {
  if (!pumping) {
    digitalWrite(PUMP, RELAY_ACTIVE_LOW ? LOW : HIGH);
    pumping = true;
  }
}

void pumpOff() {
  digitalWrite(PUMP, RELAY_ACTIVE_LOW ? HIGH : LOW);
  pumping = false;
}
The pumping flag prevents redundant digitalWrite calls on every loop iteration while already pumping, reducing unnecessary state changes.

Full Source Code

FireBot_R1.ino 
// FireBot R1 — Autonomous Fire Fighting Robot
// Arduino UNO R3
// Combines obstacle avoidance + fire extinguishing

#include <Servo.h>

const int FLAME_L = A0, FLAME_C = A1, FLAME_R = A2;
const int TRIG = 9, ECHO = 10;
const int SERVO_PAN = 6, SERVO_AIM = 5;
const int IN1=2, IN2=4, IN3=7, IN4=8, ENA=11, ENB=3;
const int PUMP = 12;

const int  FLAME_THRESH    = 500;
const int  OBSTACLE_CM     = 20;
const int  EXTINGUISH_CM   = 15;
const int  SPD             = 170;
const int  TURN_SPD        = 140;
const unsigned long BACK_MS = 300;
const unsigned long TURN_MS = 550;
const bool RELAY_ACTIVE_LOW = true;

Servo panServo, aimServo;
bool pumping = false;

void setup() {
  Serial.begin(9600);
  pinMode(IN1,OUTPUT); pinMode(IN2,OUTPUT);
  pinMode(IN3,OUTPUT); pinMode(IN4,OUTPUT);
  pinMode(ENA,OUTPUT); pinMode(ENB,OUTPUT);
  pinMode(TRIG,OUTPUT); pinMode(ECHO,INPUT);
  pinMode(PUMP,OUTPUT);
  pumpOff();
  panServo.attach(SERVO_PAN); aimServo.attach(SERVO_AIM);
  panServo.write(90); aimServo.write(90);
  stopMotors();
  delay(1000);
  Serial.println("Fire Fighting Robot Ready");
}

void loop() {
  bool fL=(analogRead(FLAME_L)<FLAME_THRESH);
  bool fC=(analogRead(FLAME_C)<FLAME_THRESH);
  bool fR=(analogRead(FLAME_R)<FLAME_THRESH);
  long d = getDistance();

  Serial.print("L:"); Serial.print(analogRead(FLAME_L));
  Serial.print(" C:"); Serial.print(analogRead(FLAME_C));
  Serial.print(" R:"); Serial.print(analogRead(FLAME_R));
  Serial.print(" D:"); Serial.println(d);

  if (!fL&&!fC&&!fR){ pumpOff(); stopMotors(); return; }
  if (d <= EXTINGUISH_CM){ stopMotors(); aimNozzle(fL,fC,fR); pumpOn(); return; }
  if (d < OBSTACLE_CM){ pumpOff(); avoidObstacle(); return; }
  pumpOff();
  if(fC||(fL&&fR)) moveForward();
  else if(fL) turnLeft();
  else if(fR) turnRight();
}

void moveForward(){ analogWrite(ENA,SPD); analogWrite(ENB,SPD); digitalWrite(IN1,HIGH); digitalWrite(IN2,LOW); digitalWrite(IN3,HIGH); digitalWrite(IN4,LOW); }
void moveBackward(){ analogWrite(ENA,SPD); analogWrite(ENB,SPD); digitalWrite(IN1,LOW); digitalWrite(IN2,HIGH); digitalWrite(IN3,LOW); digitalWrite(IN4,HIGH); }
void turnLeft(){ analogWrite(ENA,TURN_SPD); analogWrite(ENB,TURN_SPD); digitalWrite(IN1,LOW); digitalWrite(IN2,HIGH); digitalWrite(IN3,HIGH); digitalWrite(IN4,LOW); }
void turnRight(){ analogWrite(ENA,TURN_SPD); analogWrite(ENB,TURN_SPD); digitalWrite(IN1,HIGH); digitalWrite(IN2,LOW); digitalWrite(IN3,LOW); digitalWrite(IN4,HIGH); }
void stopMotors(){ analogWrite(ENA,0); analogWrite(ENB,0); digitalWrite(IN1,LOW); digitalWrite(IN2,LOW); digitalWrite(IN3,LOW); digitalWrite(IN4,LOW); }

long getDistance(){ digitalWrite(TRIG,LOW); delayMicroseconds(2); digitalWrite(TRIG,HIGH); delayMicroseconds(10); digitalWrite(TRIG,LOW); long d=pulseIn(ECHO,HIGH,25000); return d==0?999:d/58; }

void avoidObstacle(){ stopMotors(); delay(150); panServo.write(160); delay(400); long lD=getDistance(); panServo.write(20); delay(400); long rD=getDistance(); panServo.write(90); delay(200); moveBackward(); delay(BACK_MS); stopMotors(); delay(100); if(lD>=rD)turnLeft(); else turnRight(); delay(TURN_MS); stopMotors(); }

void aimNozzle(bool l,bool c,bool r){ aimServo.write(c?90:(l&&!r)?135:(r&&!l)?45:90); delay(250); }

void pumpOn(){ if(!pumping){ digitalWrite(PUMP,RELAY_ACTIVE_LOW?LOW:HIGH); pumping=true; Serial.println("PUMP ON"); } }
void pumpOff(){ digitalWrite(PUMP,RELAY_ACTIVE_LOW?HIGH:LOW); pumping=false; }
